# Reventa-Tenis-Proyecto
1. Descripción del proyecto
Nombre del proyecto
Tienda de Venta de Tenis
Descripción general
El proyecto consiste en una página web dedicada a la compra y venta de tenis. En la plataforma existirán dos tipos de usuarios:
•	Compradores: podrán registrarse, iniciar sesión, agregar productos al carrito y realizar compras.
•	Vendedores: podrán registrarse, iniciar sesión y publicar tenis para su venta.
Cada producto tendrá información detallada como marca, modelo, color, talla, precio, condición e imágenes. Los usuarios podrán buscar tenis utilizando filtros por marca, color, precio y talla.
Problema que resuelve
Actualmente, muchas personas desean vender o comprar tenis originales nuevos o seminuevos, pero no cuentan con una plataforma organizada para publicar y encontrar productos de forma rápida.
Este sistema resuelve ese problema al ofrecer una plataforma donde:
•	Los vendedores pueden publicar sus productos.
•	Los compradores pueden buscar tenis fácilmente.
•	Los usuarios pueden agregar productos al carrito.
•	Se registran las ventas realizadas.
