# 👟 Reventa-Tenis-Proyecto

## 📌 Nombre del Proyecto
**Tienda de Venta de Tenis**

---

## 📖 Descripción General

Este proyecto consiste en una página web dedicada a la compra y venta de tenis originales, nuevos y seminuevos.

La plataforma contará con dos tipos de usuarios:

### 🛒 Compradores
- Registrarse e iniciar sesión.
- Buscar productos.
- Agregar tenis al carrito.
- Realizar compras.

### 🏷️ Vendedores
- Registrarse e iniciar sesión.
- Publicar tenis para su venta.
- Administrar sus productos.

---

## 👟 Información de los Productos

Cada producto incluirá información detallada como:

- Marca
- Modelo
- Color
- Talla
- Precio
- Condición
- Imágenes

Además, los usuarios podrán buscar tenis utilizando filtros por:

- Marca
- Color
- Precio
- Talla

---

## ❗ Problema que Resuelve

Actualmente, muchas personas desean vender o comprar tenis originales nuevos o seminuevos, pero no cuentan con una plataforma organizada para publicar y encontrar productos de manera rápida y sencilla.

Este sistema busca solucionar ese problema ofreciendo una plataforma donde:

- ✅ Los vendedores pueden publicar sus productos.
- ✅ Los compradores pueden encontrar tenis fácilmente.
- ✅ Los usuarios pueden agregar productos al carrito.
- ✅ Se registran las ventas realizadas.

---

## 🚀 Objetivo del Proyecto

Desarrollar una plataforma web moderna, intuitiva y segura para facilitar la compra y venta de tenis entre usuarios.
